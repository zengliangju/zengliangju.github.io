    藏在歌曲中的记忆
    2020/05/02

    音乐承载了很多成长时期的生活记忆。听起某一首歌曲，曾经的生活画面依旧清晰地映入脑海。那些故事，那些时光，历历在目。

    “会呼吸的痛”
    ———2009年听到梁静茹这首歌，就会想起刚上高一的时候，那时候军训，第一次离家那么远，那时住在宿舍六楼，宿舍对面是一条长长的铁路轨道，每天晚上火车都会“咣咣咣”地在宿舍对面经过，火车那一排的车窗像一个个的火柴盒，看得到很多盒子都在发光，看不清车内的人。我那时经常会想一个问题，这群人从哪来，现在要去哪个城市呢？每天清晨，睡得还正香时，一首响彻楼道的歌曲作为我们整栋楼女生的起床闹钟，那时听得最多的便是梁静茹那首“会呼吸的痛”。

    “那些年”
    ———2012年这是高三时候从学校中午的广播里听得最多的一首歌，那时候我还没看过陈妍希的那部电影，但也很喜欢这首歌。那时候，正青春。内心对高考充满了期待和害怕，很矛盾。那时候觉得快要毕业了，很不舍我的同学们。那时候18岁，高考完的那个晚上是我唯一一次在网吧通宵的夜晚，那天很多同学都去了网吧，爸爸妈妈唯一一次允许我晚上跟着一群同学在外边玩。我对网吧没感觉，但当时有一种终于结束了高考的释放感，轻松自在。

    “属于”
    ———2012年，这首歌是大一时候我们班与大二学长学姐联谊时我鼓足了勇气才唱的一首歌。呼和浩特的冬天冷得非比寻常，但白天里走在雪地上的感觉很奇妙，厚厚地雪被靴子踩得发出“咯吱”声。2012年那个冬天是大学四年经历过最冷的一个冬天，最冷时有-26度。在那里，经历了很多奇妙的事情，发现冰淇淋可以吃一半放到窗外，下次再吃。从澡堂出来不吹干头发，回宿舍的路上会发现头发结冰。原来，冬天可以在湖面上走路。有一天我会喜欢上焖面、酿皮，会给室友取很多外号，吃着室友帮买的早餐，踩着点上早晨的课。

    “天空之城”
    ———2015年，大四上学期，考完研，数学考得很糟，期望落空了，觉得自己一年的辛苦白费了，很绝望，低落沮丧。我躺在宿舍床上两天，除了洗漱以外，没出门，我室友大约怕我饿死了，就往我那里扔牛肉干和奶豆腐，现在想想那几天没吃饭也没饿死，主要还是我这位室友在用零食给我补充体力。那时候我重新看了一遍“千与千寻”，又看了一遍“天空之城”。

    “毕业歌”
    ———“你总说毕业遥遥无期，转眼就各奔东西”。2016年，大四毕业，宿舍人哭着笑着一个个地在学校西门车站那里拥抱，彼此告别。送走了一个又一个室友，最后是两个室友送我，我上了校车以后，还是没忍住稀里哗啦。

    “菊次郎的夏天”
    ————离开学校去了厦门，做起了暑假工，工作简单枯燥，生活乏味，第一个月超级穷。但还是买了两本书作为晚上下班后枯燥生活的调味品，记得其中一本是“偷影子的人”，看完以后离开厦门的时候送给了组上的小伙伴。那时候常常犹豫要不要去读书。因为一分之差，调剂去了二区学校，那时候我有点想去算命，感觉命运在捉弄我。那时候我常常听“菊次郎的夏天”，常常听学长聊起他读研的生活，让我对读研又有一点向往。那时与组上的一些小伙伴玩得很开心，但到了八月份，还是辞工，去了新的学校报到。

    “送别”
    ———“天之涯，地之角，知交半零落”。2018年，我的师兄和我的师姐研三毕业。自大四毕业以后，虽然进入新的学校，却完全没了大学时期那种感觉，觉得毕业过一次以后，对毕业不会有什么感觉，但看着他们走的时候，还是觉得挺不舍。缘分这东西很奇妙，这些生命中遇见过的人，在记忆力里留下了浓墨重彩的一笔。遇见便是一种缘分。

    “盗将行”
    ———2019年8月，这是我曾单曲循环最多次的一首歌。入职，认识了一个女生，让我觉得很开心。约好做邻居，再一同去佛山培训。后来在佛山，她生病了窝在宾馆，她那些天一直闹肚子，但是是吃了一个几口西瓜才闹的肚子，而那个西瓜是我点的外卖，我给她吃的，我没和她说啥，内心却很内疚，虽然我自己吃了西瓜没啥反应。日常点滴让我感觉她是一个很nice很温暖的娃。
